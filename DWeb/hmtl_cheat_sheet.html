CSS Cheat Sheet

1. border

La propiedad border define el estilo, ancho y color del borde de un elemento.

Este div tiene un borde definido por la propiedad border.

2. border-left, border-right, border-top, border-bottom

Estas propiedades definen el estilo, ancho y color de un borde específico (izquierdo, derecho, superior o inferior) de un elemento.

Este div tiene bordes izquierdo, derecho, superior e inferior definidos individualmente.

3. border-radius

La propiedad border-radius define el radio de las esquinas redondeadas de un elemento.

Este div tiene esquinas redondeadas definidas por la propiedad border-radius.

4. background-image

La propiedad background-image define una imagen de fondo para un elemento.

Este div tiene una imagen de fondo definida por la propiedad background-image.

5. color

La propiedad color define el color del texto de un elemento.

Este texto tiene un color definido por la propiedad color.

6. font

La propiedad font es una propiedad abreviada que define varios estilos de fuente a la vez.

Este texto tiene un estilo de fuente definido por la propiedad font.

7. font-family

La propiedad font-family define la familia de fuentes para el texto de un elemento.

Este texto tiene una familia de fuentes definida por la propiedad font-family.

8. font-style, font-weight

La propiedad font-style define el estilo de la fuente (normal, cursiva, oblicua), y font-weight define el grosor de la fuente (normal, negrita, etc.).

Este texto tiene un estilo y un grosor de fuente definidos.

9. width, height

Las propiedades width y height definen el ancho y la altura de un elemento, respectivamente.

Este div tiene un ancho y una altura definidos por las propiedades width y height.

10. list-style

La propiedad list-style define el estilo de lista (tipo de viñeta, imagen, posición, etc.).

11. margin

La propiedad margin define el margen (espacio exterior) de un elemento.

Este div tiene un margen definido por la propiedad margin.

12. padding

La propiedad padding define el relleno (espacio interior) de un elemento.

Este div tiene un relleno definido por la propiedad padding.

13. overflow

La propiedad overflow define el comportamiento de un elemento cuando su contenido desborda los límites establecidos por sus dimensiones.

Este div tiene un contenido que desborda sus límites, y el comportamiento está definido por la propiedad overflow.

14. display

La propiedad display define el tipo de renderizado de un elemento (bloque, en línea, ninguno, etc.).

Este div tiene un tipo de renderizado definido por la propiedad display.
Este span tiene un tipo de renderizado definido por la propiedad display.

15. position

La propiedad position define el tipo de posicionamiento de un elemento (estático, relativo, absoluto, fijo, pegajoso).

Este div tiene un posicionamiento estático (por defecto).
Este div tiene un posicionamiento relativo.
Este div tiene un posicionamiento absoluto.
Este div tiene un posicionamiento fijo.
Este div tiene un posicionamiento pegajoso.

16. filter

La propiedad filter define efectos visuales (como desenfoque o cambio de color) en un elemento.

Esta imagen tiene un efecto visual definido por la propiedad filter.
